# **App Name**: Vini Amaral Portfolio

## Core Features:

- Home Section: Showcase a professional avatar, a strong headline indicating expertise in full-stack and AI development, and direct links to GitHub, LinkedIn, and featured projects.
- About Section: Provide an objective overview of Vini Amaral's experience, highlighting proficiency in React, Next.js, Node.js, Go, AI integration, and the ability to structure and deploy real-world projects.
- Projects Section: Display projects in a grid of modern cards, each containing the project name, a brief description, technologies used, and links to the GitHub repository or live demo, with visual emphasis on 3-5 main projects.
- Contact Section: Include direct links to Vini Amaral's email, LinkedIn, and GitHub profiles within a simple, professional layout.
- GitHub Activity Feed: Fetch Vini Amaral's latest public GitHub activity (recent commits, new repositories, etc.) via the GitHub API and display it in a dedicated section on the homepage. Use an LLM to classify each Github action in broad, human-readable terms. Example: 'Updated documentation for X project'. This gives the website visitor real-time insights on Vini Amaral's latest activites and gives more proof of their expertise. The LLM acts as a tool to convert the Github actions into human-readable classifications.

## Style Guidelines:

- Primary color: Dark slate blue (#2C3E50) to provide a professional and developer-oriented feel.
- Background color: Very dark gray (#1E293B) to create a dark mode aesthetic by default.
- Accent color: A muted blue (#3498DB) for interactive elements and highlights.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, providing a clean and modern aesthetic. 
- Use minimalist icons for navigation and project categories.
- Employ cards with soft borders and subtle hover effects to highlight projects and information sections.
- Apply subtle transition animations for a polished user experience.
